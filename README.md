# Airline Reservation System (C++)

This is a console-based Airline Reservation System developed in C++ using
object-oriented programming concepts. The system allows users to select
domestic or international flights, choose airlines and classes, book seats,
verify passenger details, and calculate the total fare.

## Features
- Domestic and International flight selection
- Multiple airlines and destinations
- Business and Economy class options
- Passenger detail entry
- Seat booking and availability
- Verification process (Passport/Visa)
- Fare calculation
- Display booking details
- Menu-driven interface

## Concepts Used
- C++ OOP (Classes & Objects)
- Conditional statements
- Loops and switch-case
- Strings and basic validation

## How to Run
1. Compile the program:
   g++ airline_reservation.cpp -o airline
2. Run:
   ./airline

## Sample Flow
1. Select airline type (Domestic/International)
2. Enter passenger details
3. Book seats
4. Complete verification
5. Confirm and view booking details

This project is created for academic and learning purposes.
